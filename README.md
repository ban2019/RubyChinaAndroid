## RubyChinaAndroid

ruby-china.org 的第三方客户端

#### 下载链接

[**Github**](https://github.com/xialincn/RubyChinaAndroid/release/ruby-china-android.apk)

### Android 版本要求

Android 4.0+

### 功能

 * 帖子分类查看
 * 浏览某一节点下的帖子
 * 用户基本资料查看
 * 登录
 * 回复
 * 发表
 * 收藏
 * 缓存帖子列表

### 开发环境

 * JDK 1.8.0
 * Android Studio 1.2.1.1
 * Android SDK 22
